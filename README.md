# UTMB-IIS

## Description:
UTMB-IIS is a web service that generates UTM links with integration of Short.io and Google Analytics 4
to collect info about users, session, clicks.

## Installation:
To install poetry:
```bash
pip install poetry
```
To install UTMB-IIS, use Poetry:
```bash
poetry install
```
